# Deployment Instructions

## Streamlit Cloud

1. Sign up or log in to [Streamlit Cloud](https://streamlit.io/cloud).
2. Click on 'New App' to create a new application.
3. Connect your GitHub account if you haven't already.
4. Select the repository `skindiagapp` and choose the branch containing your Streamlit app (typically `main`).
5. Click 'Deploy' and wait for the build process to complete. Ensure there are no errors during the build.
6. Once deployed, Streamlit Cloud will provide you with a public URL to access your application.

## Heroku

1. Sign up or log in to [Heroku](https://www.heroku.com/).
2. Create a new Heroku app from the dashboard.
3. In the "Deploy" tab, choose GitHub as the deployment method.
4. Connect your GitHub account and search for `skindiagapp` to find your repository.
5. Select the appropriate branch to deploy and click "Deploy Branch."
6. Once deployed, Heroku will provide you with a URL to your application. Adjust any environment variables in the settings if needed.

Enjoy your deployed application!
